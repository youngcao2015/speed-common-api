###  **speed-common-api** 
```
### 项目说明
```
1. Spring MVC架构，前后端分离.
2. REST FUL风格接口,响应统一格式的数据.
3. 注解式校验.
4. 全局异常处理.
5. 灵活支持JSON和JSONP.
6. 注解+AOP日志记录.
7. 统一数据格式化,如：日期、人民币等.
8. 集成Mybatis-plus做为ORM框架.
8. 集成Redis存储Token，保证接口的安全性.  
10. 集成Ehcache缓存.
11. 集成Swagger2接口管理.

```
### 快速开始
```
1. cp ~/speed-common-api
2. mvn clean package
3. mvn jetty:run
4. http://localhost:8080/swagger-ui.html

声明:本项目仅供参考学习，如有不合理之处请大家多多指教。
